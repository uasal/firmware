To run the design example, execute the following steps (for Linux only):

1. Open a terminal window.
2. Change to the directory in which you installed the Efinity software.
3. Extract the design (tar.gz format, downloaded from the link) to the location project folder of Efinity.
4. Type the following commands to open the Efinity GUI.
> source bin/efinity_sh.sh
5. To open the design using Efinity, go to File > Open Project, choose tri_state_buf.xml and open it.
6. To view the GPIO assignment, go to Tools > Open Interface Designer.
