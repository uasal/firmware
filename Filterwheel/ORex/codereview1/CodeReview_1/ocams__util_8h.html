Function:
rfc_1071x()
Description:
Calculate an RFC-1071 checksum over the specified region. If the result is 0 ( zero ) then replace it with 0xFFFF. If the parameter 'ps' is TRUE, the checksum is initialized with the parameter 'sum' which is always a precalculated IP pseudo checksum. Else it is initialized to 0.
