void clockReadSclk ( void  )
Function:
clockReadSclk()
Description:
Atomically read the 6-byte SCLK into the global variable TheSclk. The SCLK maintains the OCAMS instrument time and is periodically synchronizes with the S/C time. This routine is only called from the main thread, never from the interrupt handler.
